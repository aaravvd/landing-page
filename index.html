My Projects

A small list of my personal projects, most done over hackathons, all of which placed me in the top 5 in the hackathon.

  • Poison AI

    Poison AI is a hackathon project made by me and a friend, Rishab Jain. It is a machine learning model trained on over 6000 data points of poisonous vs nonpoisonous plants. It is an image classification model which classifies an image as either poisonous or not poisonous, with up to 95% accurarcy

  • Clarify

    Clarify is a convolutional neural network model deployed via netlify. The front-end is made using react. The backend is the CNN which is trained on a dataset of over 15k images of people's faces, designed to clarify/unblur their faces if given a blurry image of less than 50x50 pixels, with up to 99% accuracy using ResNets.

  • CA Housing Price Model

    A random forest model which predicts the price of a house in California

  • Text Summarizer

    A Python Flask app which uses a natural language processing(NLP) HuggingFace transformers model to summarize text.
